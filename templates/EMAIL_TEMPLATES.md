   - Free for any size list

   - Built-in discovery
   - Owns the reader relationship

   - Advanced automation
   - Landing pages
   - More control

   - Markdown native
   - API access
   - Privacy-focused
   - Clean, simple

- Substack for main newsletter
- ConvertKit/Mailchimp for transactional emails (welcome, download confirmation, etc.)
- Keep them separate to avoid spam filters

**Metrics to Track:**
- Open rate (aim for 30-40%)
- Click-through rate (aim for 5-10%)
- Unsubscribe rate (below 1% is good)
- Replies (engagement indicator)

**Best Practices:**
- Always include unsubscribe link
