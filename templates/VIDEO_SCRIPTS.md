## üé§ Recording Tips

### Equipment

**Minimum:**
- Built-in screen recording (macOS: Cmd+Shift+5, Windows: Xbox Game Bar)
- Laptop microphone (for voiceover)
- Free editing software (iMovie, DaVinci Resolve)

**Recommended:**
- Screen recording software: OBS Studio, Camtasia, ScreenFlow
- USB microphone: Blue Yeti, Audio-Technica AT2020
- Editing software: Final Cut Pro, Adobe Premiere, DaVinci Resolve

### Screen Recording Settings

**Resolution:** 2560√ó1440 (export 1920√ó1080)
**Frame Rate:** 60fps
**Format:** MP4 (H.264)

**Preparation:**
1. Clean desktop (hide icons)
2. Close all apps except what you need
3. Enable Do Not Disturb
4. Set app zoom level to 100%
5. Use dark theme (easier on eyes)
6. Disable OS animations (faster recording)

### Voiceover Tips

**Recording:**
- Use a quiet room (closet with clothes = good acoustics)
- Place microphone 6-8 inches from mouth
- Use a pop filter (or improvise with a sock!)
- Record in segments (easier to re-do mistakes)
- Leave 3 seconds before/after each segment

**Editing:**
- Remove mouth noises, breaths between sentences
- Normalize audio to -3dB
- Add very gentle compression
- EQ to reduce low rumble, boost clarity
- Export as WAV or high-quality AAC

### Editing Workflow

**Basic Structure:**
1. Import screen recording footage
2. Add voiceover track (if applicable)
3. Sync voiceover with visuals
4. Trim unnecessary parts
5. Add text overlays for key points
6. Add background music (quiet!)
7. Color grade for consistency
8. Export

**Keyboard Shortcuts to Show:**
- Use a keystroke visualizer (KeyCastr for macOS, Carnac for Windows)
- Or add text overlays: "‚åò+F to search"

### Music Selection

**Where to find royalty-free music:**
- YouTube Audio Library (free)
- Epidemic Sound (subscription)
- Artlist (subscription)
- PremiumBeat (per-track)
- Incompetech (free, attribution)

**Genre:** Tech/Corporate, Uplifting, Modern
**Mood:** Professional but approachable
**Tempo:** Mid-tempo (110-130 BPM)
**Volume:** -18dB to -20dB (background only)
