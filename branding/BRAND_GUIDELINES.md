## Voice & Tone

### Brand Voice

**Characteristics:**
- **Friendly:** Approachable, not corporate
- **Confident:** We know what we're doing
- **Helpful:** Here to solve problems
- **Honest:** Transparent about limitations

**Writing Style:**
- Use contractions ("we're" not "we are")
- Address users as "you"
- Active voice over passive
- Short sentences and paragraphs
- Technical but accessible

### Tone Variations

**Marketing/Website:**
- Exciting but not hyperbolic
- "Build amazing things with Claude Code"
- NOT "Revolutionary AI-powered quantum code assistant"

**Documentation:**
- Clear and instructional
- "Click the Settings icon to configure..."
- NOT "One might consider navigating to..."

**Error Messages:**
- Helpful and action-oriented
- "Couldn't connect to API. Check your key in Settings."
- NOT "Error: API_CONNECTION_FAILED"

**Social Media:**
- Casual and engaging
- "Shipped Vim mode in v0.2! ğŸ‰"
- NOT "We are pleased to announce..."

### Writing Examples

**Good:**
```
âœ… "Making Claude Code work better"
âœ… "Your code stays on your computer"
âœ… "Let's build something awesome"
âœ… "Found a bug? We'd love to fix it"
```

**Avoid:**
```
âŒ "Leveraging AI to synergize development workflows"
âŒ "Your personal information may be stored locally"
âŒ "Initiate the development process"
âŒ "Please submit a defect report"
```

## Visual Style

### Design Principles

1. **Familiar:** VSCode-inspired, developers know this layout
2. **Clean:** Plenty of whitespace, not cluttered
3. **Consistent:** Patterns repeat, predictable
4. **Accessible:** WCAG AA compliant minimum
5. **Fast:** Perceived performance through design

**Buttons:**

## Accessibility

### Color Contrast

**Minimum ratios (WCAG AA):**
- Normal text: 4.5:1
- Large text: 3:1
- UI components: 3:1

**Check all color combinations:**
- Orange (#FF6B35) on White: âœ… 3.6:1 (large text only)
- Blue (#1E3A8A) on White: âœ… 9.9:1
- Gray 600 (#4B5563) on White: âœ… 7.6:1

### Typography

**Readable:**
- Min font size: 12px
- Body text: 14px preferred
- High contrast
- Adequate line-height (1.5)

### Interactive Elements

**Design for:**
- Keyboard navigation
- Screen readers
- Reduced motion
- High contrast mode
